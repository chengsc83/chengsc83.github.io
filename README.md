# 辯時計 Pro - 專業辯論計時系統

**辯時計 Pro** 是一個專為中文辯論賽事設計的現代化、功能豐富的網頁計時系統。無論是訓練、常規賽還是大型盃賽，本系統都能提供穩定、專業且易於操作的計時與輔助功能。

## ✨ 核心功能

- **多樣化賽制支援**：
  - **新式奧瑞岡**：內建多種時間組合（三三三、四四四、五五四等），並區分「含結辯」與「無結辯」版本。
  - **特殊賽制**：支援如「辯革盃」等九辯位自由排序的創新賽制。
  - **分類選單**：清晰的賽制分類，讓您能快速找到所需流程。

- **高度客製化**：
  - **流程編輯器**：可新增、刪除、編輯、拖曳排序任一賽程階段，打造完全符合需求的比賽流程。
  - **自訂準備時間**：使用者可在賽前自由設定發言前的準備緩衝時間（Grace Period）。

- **智慧輔助功能**：
  - **語音轉錄**：利用瀏覽器 Web Speech API，可將比賽過程中的發言即時轉錄為文字稿，並在賽後匯出。
  - **比賽錄音**：支援全程麥克風錄音，並在比賽結束後提供 `.webm` 格式的音檔下載。
  - **自動語音偵測**：在準備時間結束後，可自動偵測選手發言並開始計時，無需手動操作。
  - **語音提示**：自動播報賽程階段、計時提醒（一響、二響、時間到）等。

- **現代化使用者體驗**：
  - **畫中畫 (PiP)**：可將計時器以子母畫面的方式懸浮於螢幕上，方便選手聚焦於稿件或其他視窗。
  - **鍵盤快捷鍵**：支援全域快捷鍵操作（如：下一階段 `N`、暫停 `P`、全螢幕 `F` 等）。
  - **主題切換**：提供亮色與暗色兩種主題模式，適應不同環境光線。
  - **響應式設計**：介面在桌面電腦與平板上皆有良好體驗。

## 🛠️ 技術棧

本專案完全使用前端技術打造，無需後端伺服器即可運行。

- **核心**：HTML5, CSS3, JavaScript (ES6+)
- **樣式框架**：[Tailwind CSS](https://tailwindcss.com/)
- **核心 API**：
  - **Web Speech API** (`SpeechRecognition` & `SpeechSynthesis`) 用於語音轉錄與播報。
  - **MediaRecorder API** 用於音訊錄製。
  - **Picture-in-Picture API** 用於畫中畫功能。
  - **Fullscreen API** 用於全螢幕功能。
- **輔助函式庫**：[Sortable.js](https://github.com/SortableJS/Sortable) 用於流程編輯器的拖曳排序功能。

## 🚀 安裝與啟動

本專案為純前端應用，無需複雜的安裝步驟。

#### 方法一：直接開啟（適用於基本功能）

1.  複製或下載此專案的所有檔案。
2.  直接在支援現代網頁標準的瀏覽器（如 Chrome, Edge, Firefox）中開啟[辯時計](https:/chengsc83.github.io/) 。

#### 方法二：透過本地伺服器（建議，以啟用所有功能）

部分瀏覽器對於直接從本地檔案系統 (`file:///...`) 存取麥克風等敏感權限有較嚴格的限制。為了確保錄音、語音辨識等功能正常，建議透過本地伺服器來運行。

1.  確保您的電腦已安裝 [Node.js](https://nodejs.org/) 或 [Python](https://www.python.org/)。
2.  在專案的根目錄下，打開您的終端機 (Terminal)。
3.  執行以下任一指令：

    **使用 Python:**
    ```bash
    # Python 3.x
    python -m http.server
    ```

    **使用 Node.js (需先安裝 `http-server`):**
    ```bash
    npm install -g http-server
    http-server
    ```
4.  打開瀏覽器，訪問終端機中顯示的網址（通常是 `http://localhost:8000` 或 `http://127.0.0.1:8080`）。

## 📝 使用指南

1.  **賽前設定**：
    - 填寫辯題、正反方隊名。
    - 從下拉選單中選擇一個預設的比賽流程範本。
    - （可選）在「進階設定」中調整發言準備時間、是否開啟語音偵測。
    - 若要修改流程，點擊「自定義流程」進入編輯器。

2.  **開始比賽**：
    - 點擊「開始辯論」按鈕。
    - 根據畫面上的主席稿提示，使用「上一階段」、「下一階段」、「暫停/繼續」等按鈕控制賽程。

3.  **比賽結束**：
    - 當所有流程走完後，畫面會顯示下載選項。
    - 您可以選擇下載本次比賽的 `.webm` 錄音檔或 `.txt` 逐字稿。
